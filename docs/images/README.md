# Screenshots do README

Coloque aqui os três PNGs referenciados no quickstart. Suba a stack
(`docker compose up -d --build`) e capture:

| Arquivo | URL | O que mostrar |
|---|---|---|
| `chat.png` | http://localhost:8010/ui/ | O cartão de chat do cliente com uma conversa curta (ex.: pergunta sobre energia solar + resposta do agente). |
| `admin.png` | http://localhost:8010/ui/admin/ | O painel interno logado, na aba **Dashboard** (cards de métricas) ou **Conversas** (tabela). Requer `ADMIN_PASSWORD` no `.env`. |
| `widget.png` | http://localhost:8010/ui/embed-demo.html | A página de demo com o launcher flutuante do widget aberto sobre um site fictício. |

Sugestão: largura ~1000px, formato PNG. Os nomes/caminhos já estão referenciados
no `README.md`, então basta soltar os arquivos aqui.
